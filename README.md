# SpaceStacker_OldCode

Legacy Minecraft plugin codebase for stacked entities/items/spawners.

## Build

Requirements:

- Java 8
- Maven 3.6+

```bash
git clone git@github.com:Joeyoey1/SpaceStacker_OldCode.git
cd SpaceStacker_OldCode
mvn -B clean package
```

Output JAR is produced in `target/`.

## Notes

- This repository is marked as old/legacy code.
- Dependencies are mostly `provided` and expected from server/plugin runtime.
- Use this repo for maintenance fixes and archival reference.
